# AUSG-study-2018
2018년도 AUSG 스터디 내용 정리용 저장소

## 스터디 규칙
* 주 단위 스터디 진행
* 주 1회 학습내용 공유
* 공유 내용을 정리하여 깃에 저장
* 스터디별로 브랜치 생성하여 진행
* 세부적인 스터디 진행은 스터디별로 상이
* 각 스터디 폴더에 스터디 내용이 작성되어 있음
* 스터디 결과물을 사용해 겨울 세미나 시리즈 진행
* 주기적으로 밋업을 통해 스터디내용 발표
